Программа управления светодиодом для STM32F103. Компьютер передает через USB UART интерфейс
три возможные команды:
LEDOn0#000 - светодиод включен
LEDOff#000 - светодиод выключен
LEDBli#000 - светодиод мигает
В случае ошибочной команды светодиод выключен.
При интерпритации команды микроконтроллером проверяется длина пакета, нулевой байт в конце,
таймауты между символами.
Передача осуществляется на скорости 115200 бит/с со стандартными настройками COM порта через
терминальную программу.
#000 - передача нулевого символа (разделитель между пакетами)
